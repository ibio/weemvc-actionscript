weeMVC 是一种微型快速、简洁开发框架

基于 MVC 架构，目前提供AS3.0的版本。

随着 RIA 应用逐渐在互联网中凸显重要的地位，传-统的设计思想和开发模式已经很难做到快速、简洁的发满要求。

weeMVC 是一款轻量级遵循 MVC 思想设计和实现的 ActionScript? 简洁开发框架。在保持易学易用、-快速稳定的同时，weeMVC 为迅速创建 RIA 应用提供了新的途径。

主要特征：

    * 微内核设计，性能、灵活性、-扩展能力的完美融合；
    * 整合 PureMVC 与 Cairngorm 的有点进行设计；
    * 创新的控制模型，实现了数据-、行为与表现的完美分离与复用；
    * 超简洁设计，显著降低开发成本。 

官方支持：weemvc.org SVN：http://weemvc.googlecode.com/svn/trunk/
