Class Information

Description
命令类接口。

所有的 WeeMVC 命令类都要实现此接口。然而，您不必重新自己全部实现它，因为 WeeMVC 已经帮您实现好，您只需要继承以下类即可：

  • SimpleCommand--在单个命令时可以继承此类。
  • MacroCommand--需要同时执行多个命令时，可以继承此类。
