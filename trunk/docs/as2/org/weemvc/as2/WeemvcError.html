Class Information

Description
WeeMVC 错误信息和警告信息。

此类主要记录因 WeeMVC 而引起的错误和警告。

错误：

  • 多次实例化单件类

警告：

  • 重复添加已添加过的命令类、视图类、模型类
  • 重复删除已删除过的命令类、视图类、模型类
  • 使用未添加的命令类、视图类、模型类
